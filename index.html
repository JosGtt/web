react 1

Desarrollé un contador de clics en React utilizando props y el hook useState para manejar el estado del contador. La aplicación incrementa el valor cada vez que se hace clic en un botón y muestra la cantidad de clics en pantalla, demostrando el manejo de estados y la reutilización de componentes en React.

Conversor de moneda

Desarrollé un conversor de moneda utilizando JavaScript y una API para obtener tasas de cambio en tiempo real. La aplicación permite seleccionar diferentes monedas y calcular su equivalencia de forma rápida e intuitiva, con un diseño simple y funcional
